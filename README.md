## ❗ Troubleshooting

| ❌ Problem | ✅ Fix |
|---|---|
| Mods don't apply | Run as Administrator, verify game path |
| RT still slow | Lower RT resolution in RT Performance Fix, or use No RT preset |
| Shader stutters persist | Let Stutter Eliminator fully pre-compile shaders (first launch takes ~5 min) |
| VRAM overflow crash | Use lower VRAM tier in VRAM Manager, reduce texture quality |
| Game won't launch | Restore backup, re-apply mods one by one |
| Input feels delayed | Enable Input Lag Fix + disable in-game V-Sync |
| Hair/mesh glitches | Reset RE Engine Tweaks to default values |
